# mr_server

This repository contains various documentation and configuration files for home services running on my homelab server.

## Contents

- **navidrome_playlists/** - Smart playlists for Navidrome music streaming server
- **transcribe_audio/** - Audio transcription service for automatic transcription of audio files

## Purpose

This repository serves as a centralized location for managing configurations, documentation, and automation scripts for various self-hosted services in my home environment.
